# FinTrack API

API REST para gestão financeira (receitas, despesas e relatórios mensais).

## Tecnologias
- Java 25
- Spring Boot 3.5.7
- Spring Data JPA
- H2 Database
- Maven

## Como executar localmente

mvn spring-boot:run

Acesse:

Swagger (depois): http://localhost:8080/swagger-ui

H2 Console: http://localhost:8080/h2-console

Actuator: http://8080/actuator/health
